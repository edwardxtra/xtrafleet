# **App Name**: FleetConnect

## Core Features:

- User Registration: Allow owner-operators to register and create profiles with driver and load information.
- Load Input: Enable owner-operators to input available loads, including details such as destination, weight, and preferred driver qualifications.
- Driver Input: Enable owner-operators to input drivers, including location, type of load the driver can carry, and hours of availability.
- Load Matching Tool: Employ AI to automatically match available drivers with suitable loads based on location, driver qualifications, availability, and load requirements.
- Subscription Management: Handle recurring $50 monthly subscription fee for owner-operators.
- Per-Match Billing: Record and process the $20 fee per successful driver-load match.
- Map Display: Visually show load and driver locations.

## Style Guidelines:

- Primary color: Strong blue (#2962FF), conveying reliability and trust. This color reflects the logistical and professional nature of the application.
- Background color: Light grey (#F0F2F5) for a clean, professional look, maintaining focus on the information displayed.
- Accent color: Vibrant orange (#FF7700) to highlight CTAs and important information.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, paired with 'Inter' (sans-serif) for body text.
- Use clear, vector-based icons related to transportation, location, and driver profiles.
- Implement a grid-based layout with clear sections for loads, drivers, and matching suggestions.
- Subtle transitions and loading animations to enhance user experience during load matching and data updates.
